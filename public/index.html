###小程序登录接口[app -> Manage Server]:

访问链接: https://lrs-tt.7955.com/wx/user/login 协议发送类型: POST

Request

{
	'js_code':string,//必须
	'nickName':'',	//userInfo.nickName
	'avatarUrl':'',//userInfo.avatarUrl
	'gender':'',  // userInfo.gender 性别 0：未知、1：男、2：女  ,可选
	'province':'',	//userInfo.province ,可选
	'city':'',     //userInfo.city,可选
    'country':'',	  //userInfo.country,可选
    'longitude':'',//numeric 经度
    'latitude':''//numeric  维度
}

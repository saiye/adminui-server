###扫码进游戏[app -> Manage Server]:

访问链接: https://lrs-tt.7955.com/wx/user/scene 协议发送类型: POST

Request

header
{
    "token": string, // token
}

params{
	'scene':string,//必须，扫码场景，带过来
}

其他说明:
scene值如:id=1&t=1&env=production&time=123
考虑到测试环境需要扫码功能，需要前端获取scen的env参数做下面逻辑处理，战绩，游戏中等接口考虑做相应切换.(如实现上有困难，考虑不做)
env:local,提交到 http://192.168.1.71/wx/user/scene
env:test,提交到，http://test-lrs-tt.7955.com/
env:producttion ,提交到， https://lrs-tt.7955.com/wx/user/scene

Response

{
    "code": ERetCode, // 返回码
    "errorMessage": string, // [可选] 错误描述
}
